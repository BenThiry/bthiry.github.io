Review Cards

What does HTML stand for?
HyperText Markup Language
What is the purpose of the HTML <head> element?
It contains meta-information about the HTML document
What is the HTML tag for creating a hyperlink?
<a> (anchor)
Explain the purpose of the HTML <img> tag.
It is used to embed images in the HTML document
What is the HTML tag for creating a line break?
<br>
What is the purpose of the HTML <div> element?
It is a container for other HTML elements
Explain the role of the HTML <form> tag.
It is used to create an HTML form for user input
What is the HTML tag for creating a bulleted list?
<ul>
